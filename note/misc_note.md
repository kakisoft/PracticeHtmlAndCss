## User Agent Styleheet
ブラウザに、標準のスタイルシートが組み込まれている。  
ブラウザごとに実装が異なるため、外観の統一化を図るため、リセットする事がある。

## reset.css
全部クリア（h1を使っても太字にならない、等。）
  
## normalize.css
基本的な部分を残し、クリア。
